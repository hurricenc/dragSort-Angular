## description

```
基于Angular-directive的拖拽排序实现
```

## key words

```
dragSort, Angular, drective
```

## dependency

```
node@8.11.1, @angular/cli@6.1.5
```

## you can run it

```
git clone https://github.com/qin-ziqi/dragSort-Angular.git

cd dragSort-Angular

npm install

ng serve
```

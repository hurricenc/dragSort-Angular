## description
```
基于Angular-directive的拖拽排序实现
```

## key words
```
dragSort, Angular, drective
```

## you can run it
```
git clone https://github.com/qin-ziqi/dragSort-Angular

cd dragSort-Angular

npm install  

ng serve 
```
